# 《野猪乱打手枪》1.0.0 正式发布

感谢等待——首个稳定版本现已上线。

《野猪乱打手枪》是一款用 Godot 4 制作的俯视角射击肉鸽（Roguelite）：在竞技场里抵御一波波敌人，波次之间进入商店与构筑，用掉落材料购买武器、数值强化与被动；每局选择决定角色与成长路线，单局内逐步变强，死亡后开启全新一局。

## 本版本亮点

- 波次生存：敌人随关卡推进变强、种类增多（含远程、冲锋、护盾与 Boss 等）。
- 构筑深度：升级与商店影响伤害、暴击、生命回复与元素（燃烧、冰冻减速、毒素、感电易伤等）。
- 多武器与近战：多种武器组合，配合穿透、霰弹等机制。
- 本机存档：支持本地存档与局内进度（详见仓库 `scripts/save/`）。
- 多平台：可从源码用 Godot 4.6 导出桌面 / Web / Android（导出说明见 `docs/EXPORT.md`）。

# Wild Pig Gun 1.0.0 — Initial Release

Thanks for your patience — the first stable release is here.

Wild Pig Gun (野猪乱打手枪) is a top-down shooter roguelite built with Godot 4: survive arena waves, visit the shop and build between waves, and spend materials on weapons and stat upgrades. Each run is shaped by your choices; you grow stronger during the run, and when you fall you start fresh.

## Highlights in 1.0.0

- Wave survival with scaling difficulty and enemy variety (ranged, chargers, shielded foes, bosses, and more).
- Build depth via upgrades and shop items affecting damage, crits, HP regen, and elements (burn, slow, poison, shock vulnerability, etc.).
- Weapons & melee with multiple loadouts, pierce, pellet spread, and more.
- Local save data for persistence across sessions (see `scripts/save/` in the repo).
- Multi-platform exports from source with Godot 4.6 — desktop, Web, and Android (see `docs/EXPORT.md`).
